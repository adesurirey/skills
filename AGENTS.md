# Agent guide — skills

This repository is the **source of truth** for the skills. The copies under
`~/.claude/skills/`, `~/.cursor/skills/`, and other agent dirs are **install
targets** generated by `npx skills add` — they are not version-controlled and
get overwritten on every reinstall.

## Golden rule: always edit the skill in THIS repo

When asked to create or edit a skill while working in this repo, edit the file
at `skills/<name>/SKILL.md` **inside this repo**.

**Never** edit the installed copy (e.g. `~/.claude/skills/<name>/SKILL.md` or
`~/.cursor/skills/<name>/SKILL.md`). Edits there are lost on the next
`npx skills add` and never make it into version control.

If you spot a path under `~/.claude/`, `~/.cursor/`, `~/.codex/`, or `~/.agents/`
for a skill that also exists here, that's the wrong target — switch to the
repo's `skills/<name>/SKILL.md`.
